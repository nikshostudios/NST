# Idea Browser

## What it is

An MCP-connected business context management tool. Connects directly to Claude Code so your ICP positioning, offer definition, talk tracks, competitive analysis, and growth strategy are always available as structured files. Has built-in skills for common GTM tasks (lead magnet generation, landing page architecture).

## Key feature

The business context compounds over time — each session builds on what came before rather than starting from scratch. Has an activity streak to encourage consistent iteration. Growth strategy, analytics results, and AB test learnings can all be saved back as context files.

## Where it fits

Part of the [[Wiki/concepts/AI-GTM-Stack]]. Layer 1 (context/strategy) of the Idea Browser → Paper → Humbalytics pipeline.
