## On every new session, before you do anything else

1. Read [[mi]] to learn **who** you are working with and how.
2. Read [[Home]] — the Launchpad. It tells you what's hot right now and where to look next.
3. Read [[AIOS/vault-map]] only if you still need orientation after Home.
4. If the task is a known recurring process, check [[AIOS/skills-map]] for the matching skill file.
5. If you're about to answer a business question, check [[Wiki/hot]] and [[Wiki/index]] before searching from scratch — the answer is probably already compiled.

## Golden rules

- **File over AI.** The vault files are the source of truth. You are one of many tools that reads them. Do not create state that only you can read.
- **Atlas is authored. Wiki is compiled.** Never edit a Wiki file by hand except via the `ingest-source` or `lint-wiki` skills. Atlas, Calendar, and Efforts are edited freely.
- **Raw is sacred.** Never modify files inside `Raw/`. They are the source documents. Extract from them, link to them, but do not rewrite them.
- **Every AI-generated note gets a `generated-by` frontmatter field** so the human can always tell what is theirs and what is yours.
- **When in doubt, ask, don't guess.** This vault describes a real business with real money at stake.
